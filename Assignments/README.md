# CSC574 - Web Dynamic Programming Assignments

This folder contains my practical exercises, mini-projects, and programming assignments completed throughout the **CSC574: Web Dynamic Programming** course.

The projects demonstrate my understanding of PHP, HTML, CSS, JavaScript, MySQL, sessions, cookies, authentication, and CRUD operations.

---

## Technologies Used

- PHP
- HTML5
- CSS3
- JavaScript
- MySQL
- SQL
- XAMPP

---

## Assignment Overview

| Project | Description |
|---------|-------------|
| PHP Introduction | My first PHP program and basic PHP syntax. |
| Questionnaire | HTML form containing 20 questions using text fields, radio buttons, checkboxes, and select elements. |
| Form Validation & Event Handling | JavaScript form validation using events such as `onclick`, `onfocus`, `onblur`, `onkeyup`, and `onsubmit`. |
| Cookies | Demonstrates how PHP cookies are created, read, and deleted. |
| PHP Session Exercise | Demonstrates PHP session management, login, logout, and protected pages. |
| Login System | User authentication system with registration, login, logout, and MySQL database integration. |
| PHP Connect to Database | Demonstrates connecting PHP applications to a MySQL database and inserting records. |
| CRUD | Complete Create, Read, Update, Delete application using PHP and MySQL. |
| Website | A simple multi-page PHP website demonstrating navigation between pages. |
| JS Practice | JavaScript practice covering event handling and form validation examples. |

---

## Learning Outcomes

Throughout these assignments, I learned how to:

- Develop dynamic web applications using PHP.
- Connect PHP applications with MySQL databases.
- Implement CRUD operations.
- Build user authentication systems.
- Use PHP Sessions and Cookies.
- Validate user input using JavaScript.
- Design multi-page websites.
- Handle HTML forms securely.
